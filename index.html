CSS Grid

Fully customizable, simple and responsive CSS grid

Introduction to SimpleGrid

Super simple (and flexible) grid that does what it is designed to. It's based on the ever popular 12 column grid system. Fast loading and designed to be intuitive. Feel free to take a look around. I've used it in on many projects, but more importantly I hope it will be helpful to your next project and the community as well. Please share in the comments where you use it and if you have any ideas on how it could be improved.

Made by designer & developer Jonathan Whiting - machiine.com

Features

  • Responsive, mobile first approach
  • Lightweight at 5kb
  • Supports ie8+ and all modern browsers
    chrome, firefox, safari, opera, etc
  • Nestable
  • Fluid width layout (percentages)
  • With or without gutters

Example Code

<div class="container"> <!-- Screen Width -->
  <div class="row">
    <div class="span-1">span 1 column</div>
    <div class="span-11">span 11 columns</div>
  </div>
</div>

Breakpoints

When using Responsive Grids, you can control how the grid behaves at specific breakpoints by adding class names. In other words you can control the size you want your divs to span at specific screen sizes. SimpleGrid's default responsive grids comes with the following class names and media query breakpoints.

Key CSS Media Query Applies Classname
None None Always .span-*
sm @media screen and (min-width: 0em) Always .span-*-sm
md @media screen and (min-width: 48em) ≥ 768px .span-*-md
lg @media screen and (min-width: 64em) ≥ 1024px .span-*-lg
xl @media screen and (min-width: 80em) ≥ 1280px .span-*-xl

Final Comments

This grid uses ems for the media queries. Lyza Gardner wrote an excellent article The EMs have it: Proportional Media Queries FTW! where she demonstrates the power of EMs over pixels.

Rather than optimizing for a set pixel it optimizes for the font size of a device and makes pages much more readable. It's future friendly.

The most common default browser font size is 16px, so the following equation is useful for determining 1 em's width.

100% = 1 em ~= 16px ~= 14pt
